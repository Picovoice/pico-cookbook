# Speech-to-Speech Translation

Real-time speech-to-speech translation powered by on-device AI. Speak in one language and hear it translated in real
time into another.

[![Real-Time Speech-to-Speech Translation](https://img.youtube.com/vi/D9id2Dgv_OM/0.jpg)](https://www.youtube.com/watch?v=D9id2Dgv_OM)

## Components

- [Bat Spoken Language Identification](https://picovoice.ai/docs/bat/)
- [Cheetah Streaming Speech-to-Text](https://picovoice.ai/docs/cheetah/)
- [Zebra Translate](https://picovoice.ai/docs/zebra/)
- [Orca Streaming Text-to-Speech](https://picovoice.ai/docs/orca/)
